# Credit-Card-Fraud-Detection-Analysis

This project analyzes credit card transaction data to identify fraud patterns and evaluate fraud risk using Power BI. It features dashboards that show fraud types, risk levels, monthly fraud trends, and state-wise patterns.

**Key Features**
* Fraud Rate %, Total Fraud Amount, Critical Risk Transactions
* Fraud amount by fraud type and transaction category
* Fraud distribution by risk level
* Fraudulent transactions by states

**Monthly fraud analysis**

* Filter panel for Fraud Type, State, Merchant Name
* Tools Used
* Power BI
* Excel / CSV
* DAX
* Power Query

**Insights**

* "Card Not Present" is the most frequent fraud
* December has the highest fraud counts
* Maharashtra and Karnataka show high occurrences

**How to Use**

* Download the .pbix file
* Open in Power BI Desktop
* Explore trends using the filter panel
